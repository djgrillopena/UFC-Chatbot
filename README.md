# UFC-Chatbot
Text-based chatbot created using Python

The corpus of the chatbot is a file named 'ufcchatbot.txt'

This chatbot is capable of answering questions related to the pound for pound rankings, every weight division, as well as detailed record of athletes in this organization.

Work in progress, very basic chatbot, currently working on designing an interactive user interface, as well as training the bot to respond to more questions with 
better quality of information
